
# Functions

## 1. Definition

```

define function !!name(param1, param2) is type
statements
endfunction

```

---

## 2. Invocation

```

!x = !!name(10, 20)

```

---

## 3. Return

```

return expression

```

---

## 4. Behavior

- New scope is created
- Parameters are bound to arguments
- Execution stops on return

---

## 5. Type Enforcement

Return value must match defined type.
